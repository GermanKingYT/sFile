What is sFile?

sFile is a simple file host built to be fast, portable, and lightweight.

How does it work?

Simply download the sFile executable, place it within the directory you want to share files from, and then sFile will automatically create a publicly accessible web access point for others to download files from. sFile works via commandline and executable, with option CLI arguments and a server interaction interface.
